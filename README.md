# 2048

Juego 2048, desarrollado como parte del bootcamp de globant.

[20251203-1033-40.3791281.mp4](video/20251203-1033-40.3791281.mp4)

Para levantar el proyecto tenemos que hacer los siguientes comandos

```bash
cd ex00/ && make build && make up
```

Despues accederemos a [localhost:5555](http://localhost:5555) y ya se podra jugar el objetivo del juego es combinar las celdas de tienen el mismo valor para alcalzar la puntuacion de 2048, funciona con la cruceta del teclado (←↑→↓)

Cuando queramos dejar de jugar hacemos el siguiente comando para apagar el docker

```bash
make down
```
